# 🩺 Diabetes Prediction App

This project is a **Machine Learning based web application** that predicts whether a person is likely to have diabetes based on medical parameters.

The model is trained using the **PIMA Indians Diabetes Dataset** and uses machine learning algorithms to analyze patient health data.

---

# 📌 Project Overview

Diabetes is a chronic disease that affects millions of people worldwide. Early prediction can help in timely treatment and prevention.

This project uses **Machine Learning techniques** to predict the probability of diabetes based on health-related features such as:

- Glucose Level
- Blood Pressure
- BMI
- Insulin
- Age
- Skin Thickness
- Diabetes Pedigree Function

---

# 🧠 Machine Learning Workflow

The following steps were followed in building the model:

1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Feature Scaling
5. Model Training
6. Model Evaluation
7. Deployment

---

# 📊 Dataset

Dataset used in this project:

**PIMA Indians Diabetes Dataset**

Features in the dataset:

| Feature | Description |
|------|------|
Pregnancies | Number of times pregnant |
Glucose | Plasma glucose concentration |
BloodPressure | Diastolic blood pressure |
SkinThickness | Triceps skin fold thickness |
Insulin | 2-Hour serum insulin |
BMI | Body mass index |
DiabetesPedigreeFunction | Genetic influence |
Age | Age of the patient |

Target Variable:

- **Outcome**
- 0 → No Diabetes
- 1 → Diabetes
